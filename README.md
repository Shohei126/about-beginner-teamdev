# チーム開発全体のスケジュール

スケジュールについては参考例であり、デプロイ以外のチェックリストのタスクを１周目（4/13）までに完了していれば構いません。

```

8/10: チームのリポジトリの作成、チームで自習室に集まりアイスブレイクと課題を行う日程を決める、個人でのGit学習
8/11: 個人でのGit学習
8/12: 自習室に集まり、課題1を着工（1回目）
8/13: 個人でのGit学習
8/14: 自習室に集まり、課題1を着工（2回目）
8/15: チームで作成する成果物を決定
8/16: チームメンバーが課題の内容を理解している状態、チェックリスト（リポジトリ）の提出
8/17: チーム開発着工
~~
8/25: 成果物提出


```
